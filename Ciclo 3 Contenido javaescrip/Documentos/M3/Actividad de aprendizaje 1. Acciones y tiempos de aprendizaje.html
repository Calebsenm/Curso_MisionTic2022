Conoce las generalidades de Javascript: definición, declaración de variables y operadores.

Acciones de aprendizaje

1. Conoce qué es Javascript, así como definición y uso de variables y operadores en este lenguaje de programación, viendo los diferentes recursos:

  1.  ¿Qué es javascript? con Xaca Rana. (2021).  01 - Javascript 101 - ¿Qué es javascript?. [Video].
  2. Variables con Xaca Rana. (2021).  02 - Javascript 101 - Variables. [Video].
  3. Operadores y condiciones con Xaca Rana. (2021).  03 - Javascript 101 - Condición. [Video].
  4. Cliclos con Xaca Rana. (2021).  04 - Javascript 101 - Ciclos. [Video].

2. Realiza un “Hola Mundo” en javascript en el portafolio editado en la actividad anterior.

Formas de trabajo

  • Individual: el estudiante es responsable de su proceso de aprendizaje y de las actividades que realiza.
