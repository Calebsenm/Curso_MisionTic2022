Conoce sobre fuentes para descarga gratis de plantillas HTML y CSS.

Acciones de aprendizaje

1. Configura y edita una plantilla con Xaca Rana. (2021).  19 - Programación Web - HTML + CSS + Javascript (Templates). [Video].

2. Explora recursos gratuitos que pueden ser útiles para el desarrollo web:

  1. Plantillas HTML en  W3Layoutshttps.
  2. Plantillas HTML5 Bootstrap en Freehtml5.
  3. Plantillas y kits de interfaz de usuario con Uideck.
  4. Plantillas Bootstrap que son diseños CSS receptivos HTML5 en Templatemo.
  5. Plantilla HTML5 personalizable en html5up.

3. Practica la edición y configuración de una plantilla tomada de cualquier fuente compartida en la acción anterior.
