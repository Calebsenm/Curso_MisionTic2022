Aprende sobre HTML, lenguaje que permite definir la estructura del documento web.

Acciones de aprendizaje

1. Realiza una introducción a HTML Y Aprende sobre Encabezados y Etiquetas básicas en HTML:

  1. Bedoya, A. [XACA RANA]. (2021). 03 -Programación Web - Etiquetas HTML básicas y encabezados. [Video].

2.Aprende cómo subir un sitio web a un repositorio con tres paginas enlazadas:

  1. Bedoya, A. [XACA RANA]. (2021).04 - Programación Web - Repositorio. [Video].

3. Genera una url pública desde github:

  1. Bedoya, A. [XACA RANA]. (2021). 17 - Dev Tools - Github pages. [Video].

4. Conoce la anatomía de una etiqueta, block vs line, contenedores semanticos y listas numeradas o no numeradas:

    1. Bedoya, A. [XACA RANA]. (2021). 05 - Programación Web - Otras etiquetas html. [Video].

5. Conoce herramientas útiles para practicar html:

    1. Bedoya, A. [XACA RANA]. (2021). 06 - Programación Web - Multimedia. [Video].
    2. Realiza una actualización del repositorio con nueva pagina con video y audio usando rutas absolutas.

6. Profundiza en las etiquetas HTML y los formularios (parte 1):

    1. Bedoya, A. [XACA RANA]. (2021). 07 - Programación Web - Html formulario 1 / 2. [Video].

7. Profundiza en las etiquetas HTML y los formularios (parte 2):

    1. Bedoya, A. [XACA RANA]. (2021). 08 - Programación Web - Html formulario 2 / 2. [Video].
    2. Realiza los siguientes formularios usando etiquetas html: Login, registro, olvido de contraseña, ventana Actualizar usuario, ventana de perfil, formulario no editable y actualizar repositorio con formulario sin validación

8. Aprende cómo realizar un pull request:

    1. Bedoya, A. [XACA RANA]. (2019). 14 - Dev tools - Pull Request Hacktoberfest. [Video].
    2. Realiza un formulario y luego integralo por medio de pull request tus los compañeros.

Formas de trabajo

  • Individual: el estudiante es responsable de su proceso de aprendizaje y de las actividades que realiza.

Tiempos (Semana 1)

  • Trabajo autónomo del estudiante: 6 horas
  • Acompañamiento docente: 2 hora.
