Reconoce los diferentes stacks de desarrollo de aplicaciones web.

Acciones de aprendizaje

1. Revisa los siguientes recursos de apoyo y contextualiza los diferentes stacks empleados para el desarrollo de aplicaciones web:

  1.  UPB Académico. (2021, 28 de octubre). Componentes stack tecnológico I - Jefferson Peña Torres [Video].
  2. UPB Académico. (2021, 28 de octubre). Componentes stack tecnológico II - Jefferson Peña Torres. [Video].
  3. UPB Académico. (2021, 28 de octubre). Componentes stack tecnológico III - Jefferson Peña Torres [Video].

2. Lee el siguiente contenido y analiza las diferencias entre los stacks de desarrollo:

    1. Syntonize. (2021). ¿Cuál es el stack tecnológico más popular? - Syntonize. [Artículo en línea] 

3. Compara MERN con otros stacks empleados para el  desarrollo de aplicaciones web, diferentes al MEAN stack, y realiza un cuadro sinóptico en el que enuncies al menos 5 diferencias.

4. Accede a la presentación utilizada por el docente:

a. Peña, J. (2021) Componentes necesarios para construir una aplicación web. UPB. [Presentación de diapositivas]
