Aprende a usar el framework React JS en el desarrollo de aplicaciones web.

Acciones de aprendizaje

1. Adquiere los bases conceptuales sobre el framework React JS, revisa los siguientes contenidos y realiza las configuraciones:

  1. UPB Académico. (2021, 28 de octubre). Introducción a React JS - Jefferson Peña Torres [Video].
  2. UPB Académico. (2021, 28 de octubre). Componentes dentro de React JS - Jefferson Peña Torres [Video].
  3. UPB Académico. (2021, 28 de octubre). Primeros pasos con React JS - Jefferson Peña Torres [Video].

2. Crea tu primera aplicación haciendo uso de React JS, SPA con frameworks de desarrollo para aplicaciones web:

  1. UPB Académico. (2021, 28 de octubre). Creación de primera SPA (Part. 1) - Jefferson Peña Torres [Video].
  2. UPB Académico. (2021, 28 de octubre). Creación de primera SPA (Part. 2) - Jefferson Peña Torres [Video].

3. Accede a la presentación utilizada por el docente y completa tus notas de clase:

  1. Peña, J. (2021) SPA Básica. UPB. [Presentación de diapositivas]
  2. Peña, J. (2021) Primeros pasos en React JS UPB. [Presentación de diapositivas]
