Implementa la base de datos, realiza la configuración para la conexión con JAVA

Acciones de aprendizaje

1. Crea una base de datos con ayuda de XACA RANA. (2021). 52 - Curso Programación JAVA - Creación base datos. [Video].

2. Realiza la configuración y conexión a la base de datos desde JAVA / JDBC con XACA RANA. (2021). 53 - Curso Programación JAVA - Configuración a base de datos. [Video]

3. Obten datos desde JAVA e imprimirlos en un control tipo lista en la interfaz gráfica, es importante tener presente el patrón MVC. XACA RANA. (2021). 54 - Curso Programación JAVA - Carga Datos Interfaz Gráfica. [Video].

4. Crea una clave foránea con XACA RANA. (2021). 55 - Curso Programación JAVA - Clave Foranea. [Video].

Formas de trabajo

  • Individual: el estudiante es responsable de su proceso de aprendizaje y de las actividades que realiza.

Tiempos (Semana 7)

  • Acompañamiento docente: 2 horas.
