Instala y configura el gestor de base de datos.

Acciones de aprendizaje

1. Realiza la instalación y configuración del gestor base de datos MySQL: 

  1. Instalación de Laragon con XACA RANA. (2020). 01 - Programacion web - Ambiente de desarrollo. [Video].
  2. Configuración base de datos y phpmyadmin con XACA RANA. (2021). 51 - Curso Programación JAVA - Configuración de Laragon, Mysql y phpmyadmin. [Video].

2. Interactúa con la base de datos usando el gestor de phpmyadmin y realice consultas SQL, INSERT para agregar registros, UPDATE para actualizar campos y DELETE para borrar registros de prueba
