Realiza un diagrama de casos de uso y código en Java

Acciones de aprendizaje

1. Conoce sobre valores por defecto, constructores y diagramas de casos de uso, visualizando los videos:

  1. Valores por defecto de Bedoya, A. [XACA RANA]. (2021). 28 - Curso Programación JAVA - Valores por defecto. [Video].
  2. Conoce la importancia de instacia de los constructores con Bedoya, A. [XACA RANA]. (2021). 29 - Curso Programación JAVA - Constructores. [Video].
  3. observa cómo se pueden crear los diagramas de casos de uso con Bedoya, A. [XACA RANA]. (2021). 29 - Curso Programación JAVA - Diagrama de casos de uso. [Video].

2. Para las clases creadas en la actividad 1, realiza:

  1. Un diagrama de casos de uso.
  2. Agrega tres constructores en cada una de las clases.
  3. Crea nueve instancias de las clases usando sus constructores.

3. Comparte el diagrama de casos de usos, junto con el código de clases de JAVA con las modificaciones realizadas en los puntos anteriores, en: Foro diagrama de casos de uso y código en Java. Esta actividad es formativa y no tiene peso en la nota de la unidad.

Formas de trabajo

  • Individual: el estudiante es responsable de su proceso de aprendizaje y de las actividades que realiza.

Tiempos (Semana 3)

  • Trabajo autónomo del estudiante: 9 horas.
  • Acompañamiento docente: 3 horas.

Evaluación formativa:

  • Implementa adecuadamente los constructores a cada una de las clases.
  • Utiliza adecuadamente el instanciamiento de clases usando constructores.
