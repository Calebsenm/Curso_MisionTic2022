Conoce sobre las clases y objetos en Java.

Acciones de aprendizaje

1. Visualiza los videos en donde aprenderás conceptos claves en JAVA como las clases, objetos, abstracción y diagramas de clases:

  1. Aprende sobre abstracción, concepto de clase y objeto con Bedoya, A. [XACA RANA]. (2021). 25 - Curso Programación JAVA - Abstracción, clases y objetos. [Video].
  2. Conoce sobre los diagramas de clase con Bedoya, A. [XACA RANA]. (2021). 26 - Curso Programación JAVA - UML - Diagrama de clases. [Video].
  3. Aprende sobre la creación de una clase en JAVA con Bedoya, A. [XACA RANA]. (2021). 27 - Curso Programación JAVA - Creacion Clase. [Video]

2. Realiza el diseño de tres clases que representen a: Una Factura, un Producto y un Cliente, para ello ten en cuenta los siguientes contenidos:

  1. Profundiza sobre diagramas UML con UNAD. (s.f). ¿Qué es UML? Unidad 1: Lenguaje de Modelado Unificado.
  2. Aprende cómo hacer diagramas de clases UML con Lucidchart. (s.f). Tutorial de diagramas de clases UML.

3. Implementa, en JAVA, las clases diseñadas en la acción anterior. Ten en cuenta que cada clase debe cumplir con alta cohesión y bajo acoplamiento.

4. Crea una instancia de cada clase en la clase principal del programa. Asigne y modifique valores usando métodos set y get, realice preguntas usando métodos is.

5. Comparte el diagrama UML, junto con el código de JAVA, realizado en las acciones anteriores en Foro diagrama UML y uso de clases en Java. Esta actividad es formativa y no tiene peso en la nota de la unidad.

Formas de trabajo

  • Individual: el estudiante es responsable de su proceso de aprendizaje y de las actividades que realiza.

Tiempos (Semana 3)

  • Trabajo autónomo del estudiante: 12 horas.
  • Acompañamiento docente: 4 horas.

Evaluación formativa:

  • Representa de manera correcta el algorito para implementar lases por medio del uso de diagrama UML.
  • Hace un uso correcto de los simbolos del diagrama UML lo que lo que evidencia en un algoritmo con sentido.
  • Diseña código un código en Java extraido de la diag´ramacion en UML.
