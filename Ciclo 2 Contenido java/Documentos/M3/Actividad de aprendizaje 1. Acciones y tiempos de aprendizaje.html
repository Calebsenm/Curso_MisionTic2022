Crea un juego tipo Number Puzzle.

Acciones de aprendizaje

1. Visualiza los videos con los que aprenderás sobre las clases abstractas, interface, interfaz de usuario y los eventos:

  1. Clase abstractas con XACA RANA. (2021). 35 - Curso Programación JAVA - Clase Abstracta. [Video]

  2. Concepto y creación de una interface con XACA RANA. (2021). 36 - Curso Programación JAVA - Interface. [Video]

  3. Interfaz de usuario y eventos con XACA RANA. (2021). 37 - Curso Programación JAVA - Interfaz de usuario UI. [Video]

2. Crea un programa que simule el tablero de juego Number Puzzle. El espacio en blanco debe ser un botón sin texto.

3. Analiza la lógica del juego Number Puzzle, usando un tablero de 3x3.


4. Programa la interfaz del juego para que permita ordenar los números, la lógica tendrá en cuenta que el movimiento de los números se da al interactuar entre dos botones, como muestra el ejemplo.

5. Use un listener que procese el evento click sobre un arreglo de botones, correspondientes a los botones de la interfaz trabajada.

6. Realiza un video corto en donde compartas con tus compañeros el funcionamiento de tu interfaz del juego Number Puzzle. Comparte en video en Foro Interfaz juego Puzzle.

Formas de trabajo

  • Individual: el estudiante es responsable de su proceso de aprendizaje y de las actividades que realiza.

Tiempos (Semana 4)

  • Trabajo autónomo del estudiante: 9 horas.
  • Acompañamiento docente: 3 horas.

Evaluación formativa:

Crea un juego tipo Number Puzzle.
